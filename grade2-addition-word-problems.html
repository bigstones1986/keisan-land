足し算の文章題を解く3つの手順

  1. はじめにある数と、あとから増えた数を見つけます。
  2. 2つを合わせる場面なら、数を + でつなぎます。
  3. 一の位から計算し、10以上になったら十の位へ1くり上げます。

「ぜんぶで」「合わせる」「あとから増えた」という場面を確かめてから式を作ります。言葉だけで決めず、数が増えているかを見ます。

問題

  1. ノートが 24さつ あります。あとから 13さつ ふえました。ノートは ぜんぶで なんさつ ありますか。

    しき: ____ + ____ = ____　こたえ: ____さつ

  2. えんぴつを 36ほん もっています。おともだちから 8ほん もらいました。えんぴつは ぜんぶで なんぼん ありますか。

    しき: ____ + ____ = ____　こたえ: ____ほん

  3. 赤いシールが 28まい、青いシールが 17まい あります。シールは ぜんぶで なんまい ありますか。

    しき: ____ + ____ = ____　こたえ: ____まい

  4. 図書室に 本が 45さつ あります。新しく 26さつ 入りました。本は ぜんぶで なんさつ ありますか。

    しき: ____ + ____ = ____　こたえ: ____さつ

  5. 公園に 子どもが 19人 います。あとから 14人 来ました。子どもは ぜんぶで 何人 いますか。

    しき: ____ + ____ = ____　こたえ: ____人

  6. カードを 57まい 持っています。さらに 12まい もらいました。カードは ぜんぶで なんまい ありますか。

    しき: ____ + ____ = ____　こたえ: ____まい

  7. みかんが 38こ あります。りんごが 29こ あります。くだものは ぜんぶで なんこ ありますか。

    しき: ____ + ____ = ____　こたえ: ____こ

  8. 水そうに めだかが 46ひき います。あとから 7ひき 入れました。めだかは ぜんぶで なんびき いますか。

    しき: ____ + ____ = ____　こたえ: ____ひき

  9. お店に パンが 64こ あります。朝に 18こ やきました。パンは ぜんぶで なんこ ありますか。

    しき: ____ + ____ = ____　こたえ: ____こ

  10. 花だんに チューリップが 35本、ひまわりが 25本 あります。花は ぜんぶで 何本 ありますか。

    しき: ____ + ____ = ____　こたえ: ____本

答えと考え方

  1. 24 + 13 = 37　こたえ 37さつ24さつに、増えた13さつを足します。
  2. 36 + 8 = 44　こたえ 44ほん持っていた36ほんに、もらった8ほんを足します。
  3. 28 + 17 = 45　こたえ 45まい赤28まいと青17まいを合わせます。
  4. 45 + 26 = 71　こたえ 71さつもとの45さつに、新しい26さつを足します。
  5. 19 + 14 = 33　こたえ 33人いた19人に、来た14人を足します。
  6. 57 + 12 = 69　こたえ 69まい持っていた57まいに、もらった12まいを足します。
  7. 38 + 29 = 67　こたえ 67こみかん38こと、りんご29こを合わせます。
  8. 46 + 7 = 53　こたえ 53ひきいた46ひきに、入れた7ひきを足します。
  9. 64 + 18 = 82　こたえ 82こあった64こに、焼いた18こを足します。
  10. 35 + 25 = 60　こたえ 60本チューリップ35本と、ひまわり25本を合わせます。

文章題で止まったときの見守り方

  • 2つの数に線を引き、何の数かを一緒に読みます。
  • 数が増える場面か、2つを合わせる場面かを確かめます。
  • くり上がりで迷ったら、一の位だけを先に計算します。
  • 答えには「さつ」「本」「人」など、問題と同じ単位を付けます。

計算だけを復習するときは、小学2年生の足し算ドリルが使えます。数が減る場面を練習するときは、小学2年生の引き算の文章問題へ進めます。
